## 功能摘要

### 📁 素材庫
- 三個分頁：影片 (MP4)、音效 (MP3)、圖像 (PNG/JPG)
- 拖曳或點擊「+」加入時間軸

### 🎬 時間軸 (多軌道)
- **視頻軌**：放置 MP4 片段，顯示縮圖
- **文字軌**：放置文字或圖像疊加層
- **音效軌**：放置 MP3 音效
- **分割**：將播放頭移至片段上按「✂ 分割」或按 `S`
- **刪除**：選取片段後按「🗑 刪除」或 `Delete`
- 片段可**拖動移位**、兩端可**拖曳調整長度**
- 縮放滑桿控制時間軸比例

### 📝 文字功能
- 「T+ 文字」新增文字，可設定字型、大小、顏色、描邊
- 在預覽畫面上**直接拖動文字**改變位置
- 右側屬性面板精確調整 X/Y 座標

### ▶ 預覽播放
- 播放/暫停 (空白鍵)、前後跳轉、點擊時間尺拖曳定位

### ↩ 復原/重做
- `Ctrl+Z` / `Ctrl+Y`，支援最多 80 步

### 💾 專案檔 (.ipmos)
- 儲存：圖像、音效嵌入 Base64（< 8MB），影片儲存路徑參考
- 開啟：自動還原所有軌道位置與屬性

### ▶ 匯出影片
- 選擇解析度 (360p–1080p)、幀率、位元率
- 即時渲染輸出 WebM 格式（Chrome 最佳相容）
